🏥 HealthEase - Smart Healthcare System

 📌 Overview
HealthEase is a web-based healthcare management system that allows users to book doctor consultations, manage appointments, access medical records, and get AI-based health suggestions.

🚀 Features
- User Authentication (Email & OTP Login)
- Doctor Consultation Booking
- Appointment Scheduling
- AI Health Assistant Chatbot
- Emergency Services Module
- Pharmacy Section
- Health Records Management

 🛠️ Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js (if used)
- Express.js (if used)

💡 How to Run
1. Clone the repository
2. Open index.html in browser (for frontend)
3. Run backend using:
   npm install
   node server.js
